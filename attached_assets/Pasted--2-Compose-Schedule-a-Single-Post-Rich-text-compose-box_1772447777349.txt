
2. 📝 Compose & Schedule a Single Post

Rich text compose box (500 char limit with live counter)
Option to attach image or video
Two buttons: "Post Now" and "Schedule"
If Schedule is chosen, show a date-time picker
Store scheduled posts in SQLite
APScheduler runs in background and publishes at the chosen time
Show all upcoming scheduled posts in a "Scheduled Queue" list with ability to edit or cancel


3. 📦 Bulk Post (Multiple Threads in Sequence)

A bulk compose section where user can add multiple posts (click "+ Add Thread" to add more)
Each post has its own text box + optional media
User sets a delay between posts (e.g. 5 mins, 10 mins, 30 mins, 1 hour — dropdown)
A "Start Bulk Queue" button fires the first post immediately and schedules the rest using APScheduler with the set delay between each

After a post is published, user can set a follow-up reply

APScheduler posts the follow-up as a reply to the original post at the set time
Show an in-app notification: "Your follow-up was posted! Don't forget to pin it manually in the Threads app" (Threads API doesn't support pinning yet)

Refresh button to fetch latest comments

Connected account info (username, profile pic, follower count if available)

Quick shortcuts to all pages


🔌 Threads API Endpoints to Use

GET https://graph.threads.net/v1.0/me — get user profile
POST https://graph.threads.net/v1.0/{user_id}/threads — create a post container
POST https://graph.threads.net/v1.0/{user_id}/threads_publish — publish the post
GET https://graph.threads.net/v1.0/{post_id}/replies — get replies/comments
POST https://graph.threads.net/v1.0/{user_id}/threads with reply_to_id — reply to a post
POST https://graph.threads.net/v1.0/{media_id}/likes — like a comment
